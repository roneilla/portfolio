RB CREATIVE LAB info
I'm Roneilla, and this is my digital playground.
In late 2018, I started a Google Docs page called "Creative Lab". We had just recently started to learn about Arduinos and physical computing at school, and I was learning how to use A-Frame for a project. I realized then how much I loved to learn.

I spent 10 weeks learning A-Frame from scratch, and trying to go beyond by seeing how I can incorporate JavaScript into the framework in order to create an interactive VR experience.

I love to learn and explore what I can do with the combination of design and technology.

My approach to learning is to figure out the how before making the what. I like to research and read documentation while I learn in order to fully understand what I can do with it. Of course, learning wouldn't be complete without doing. After doing some research I jump into the code to try to implement what I learned, then try to go beyond the examples. I try to find examples and sample projects, and mix up the code to see what I can create. These experiments and remixes help me learn and understand the code.

I strive to create new and unique experiences that inspire others to start creating and explore the world of creative technology.
I'm always up for a chat! If you wanna talk ESP8266, P5js, or how iced coffee is superior to hot coffee, don't hesitate to reach out!
